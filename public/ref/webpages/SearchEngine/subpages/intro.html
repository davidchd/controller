Search Engine

A search engine is a web-based system that enable users to search for specific information across the internet. A search engine differs from a web directory, which is updated and maintained by humans; search engines usually use a small piece of software to loop up for new information real-time and automatically. With a query given by a user, a search engine will look for relevant information in its database and show the result to the user in certain patterns. It is to say, the most core part of a search engine might be its database and the patterns it shows the result. The approach that a search engine builds and updates its database includes two parts: crawl and index; and the patterns for a search engine to display the result is based on web pages' rank.
